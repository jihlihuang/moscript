# F04 — 字圖上傳

> 功能類別：內容貢獻
> 相關檔案：`lib/glyph-upload.ts`、`lib/blob-storage.ts`、`app/api/glyphs/upload/`、`app/upload/`、`lib/validation.ts`

---

## 1. 功能概述

登入使用者可上傳自己的書法字圖到系統。上傳時需指定中文字、書體、作者等元資料，系統自動儲存原圖並生成縮圖。上傳的字圖預設為私密，使用者可自行設定為公開。

管理員另有後台批量上傳功能（見 [F08-後台管理.md](F08-後台管理.md)），直接上傳至系統書庫。

---

## 2. 上傳限制

| 限制項目 | 值 |
|---------|-----|
| 最大檔案大小 | 20 MB |
| 速率限制 | 10 次 / 分鐘 / IP |
| 支援格式 | jpg、jpeg、png、webp、gif、bmp、ico、avif、heic、heif、tif、tiff |
| 必填欄位 | 中文字（char）、圖片檔案 |

---

## 3. 功能流程

### 3.1 一般使用者上傳流程

```
使用者至 /upload 頁面
  │
  ├── 填寫表單：
  │     ├── 中文字（必填，僅接受單一 CJK 字元）
  │     ├── 書體（選填）
  │     ├── 作者（選填）
  │     ├── 作品名稱（選填）
  │     └── 可見性（public / private）
  │
  ├── 選擇圖片檔案
  │     └── 前端驗證格式與大小
  │
  └── 提交 POST /api/glyphs/upload
        ├── 速率限制檢查（10 次/分鐘/IP）
        ├── Session 驗證（需登入）
        ├── 驗證中文字格式
        ├── 驗證檔案格式與大小
        ├── 產生安全檔案名稱
        ├── 儲存原圖
        │     ├── 公開 → public/glyphs/[char]/[filename]
        │     └── 私密 → data/private-glyphs/[char]/[filename]
        ├── 寫入 glyphs 資料表
        │     └── source = 'personal-upload', owner_user_id = 使用者 ID
        ├── 記錄 usage_event: upload_succeeded / upload_failed
        └── 同步資料庫至 Blob（如有設定）
```

### 3.2 檔案名稱生成規則

```
格式：[作者]_[書體]_[標題]_[時間戳].[副檔名]
範例：王羲之_行_蘭亭集序_1735689600000.jpg

特殊規則：
- 缺少欄位以空字串代替（前後底線會清除）
- 中文與特殊字元以 encodeURIComponent 編碼後存入路徑
- 目錄結構：[char]/[filename]
```

### 3.3 儲存路徑邏輯

| 情境 | 路徑 |
|------|------|
| 公開字圖（本機） | `public/glyphs/[char]/[filename]` |
| 私密字圖（本機） | `data/private-glyphs/[char]/[filename]` |
| 公開字圖（Azure） | Blob：`glyphs/[char]/[filename]` |
| 私密字圖（Azure） | Blob：`private-glyphs/[char]/[filename]` |

### 3.4 URL 格式

| 情境 | image_url 格式 |
|------|---------------|
| 公開字圖（本機） | `/glyphs/[char]/[filename]`（靜態檔案） |
| 私密字圖（本機） | `/api/glyphs/[id]/asset`（動態 API，驗證後回傳） |
| Azure Blob | `https://[account].blob.core.windows.net/[container]/[path]` |

## 4. API 規格

### POST /api/glyphs/upload
**用途**：上傳個人書法字圖

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要登入 |
| Content-Type | `multipart/form-data` |
| 速率限制 | 10 次/分鐘（IP 級） |

**表單欄位：**

| 欄位名稱 | 類型 | 必填 | 說明 |
|---------|------|------|------|
| `file` | File | 是 | 圖片檔案（最大 20 MB） |
| `char` | string | 是 | 中文字（單一 CJK 字元） |
| `scriptType` | string | 否 | 書體 |
| `author` | string | 否 | 書寫者名稱 |
| `workTitle` | string | 否 | 作品名稱 |
| `visibility` | string | 否 | `public` 或 `private`（預設 `public`） |

**成功回應（201 Created）：**
```json
{
  "id": 123,
  "char": "山",
  "image_url": "/glyphs/山/王羲之_行_蘭亭集序_1735689600000.jpg",
  "thumbnail_url": null,
  "visibility": "public"
}
```

**錯誤回應：**

| HTTP 狀態碼 | 情境 |
|------------|------|
| 401 | 未登入 |
| 400 | 缺少必填欄位 / 無效中文字 / 不支援的檔案格式 |
| 413 | 檔案超過 20 MB |
| 429 | 超過速率限制（10 次/分鐘） |

---

### GET /api/glyphs/[id]/asset
**用途**：讀取私密字圖檔案

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要為字圖所有者或管理員 |
| 回應格式 | 圖片二進位（image/\* Content-Type） |

---

## 5. 資料驗證規則

| 欄位 | 驗證規則 |
|------|---------|
| char | 必須為單一 Unicode CJK 字元（一-鿿 等範圍） |
| author | 最長 100 字元，截斷超出部分 |
| scriptType | 最長 50 字元 |
| workTitle | 最長 200 字元 |
| 檔案格式 | MIME type 需在允許清單中 |
| 檔案大小 | 不超過 20,971,520 bytes（20 MB） |

---

## 6. Azure Blob Storage 整合

| 設定項目 | 說明 |
|---------|------|
| 啟用條件 | 環境變數 `AZURE_STORAGE_CONNECTION_STRING` 已設定 |
| 公開字圖前綴 | `MOSCRIPT_GLYPHS_PREFIX`（預設：`glyphs`） |
| 私密字圖前綴 | `MOSCRIPT_PRIVATE_GLYPHS_PREFIX`（預設：`private-glyphs`） |
| 上傳行為 | 本機儲存後同步上傳至 Blob |
| 讀取行為 | 公開字圖直接用 Blob URL；私密字圖透過 `/api/glyphs/[id]/asset` 代理 |

---

## 7. 本機遷移機制

系統啟動時會自動檢查私密字圖是否誤存於 `public/glyphs/`，若有則自動遷移至 `data/private-glyphs/` 並更新資料庫中的 URL（`migrateLocalPrivateGlyphFiles` 函式）。

---

## 8. 相關環境變數

| 變數名稱 | 說明 |
|---------|------|
| `AZURE_STORAGE_CONNECTION_STRING` | Azure Blob Storage 連線字串（選填） |
| `AZURE_STORAGE_CONTAINER_NAME` | Blob 容器名稱（選填） |
| `MOSCRIPT_GLYPHS_PREFIX` | 公開字圖 Blob 路徑前綴 |
| `MOSCRIPT_PRIVATE_GLYPHS_PREFIX` | 私密字圖 Blob 路徑前綴 |
| `MOSCRIPT_DATA_DIR` | 本機資料目錄（預設：`data`） |
