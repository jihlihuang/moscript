# F03 — 集字作品

> 功能類別：核心集字功能
> 相關檔案：`app/api/collections/`、`app/collections/`、`app/me/`、`components/CollectionGlyphDisplay.tsx`

---

## 1. 功能概述

集字作品是 MoScript 的主要產出物。使用者從搜尋結果中選取每個字對應的書法字圖，組合成一件「集字作品」，可設定標題、顯示方向（橫排/直排），並選擇公開或私密。作品建立後可分享連結或下載。

---

## 2. 集字作品資料結構

一件集字作品包含：
- **作品基本資訊**：標題、集字文字、顯示方向、可見性、建立者
- **集字項目**：每個字對應的字圖（glyph）與位置

---

## 3. 功能流程

### 3.1 建立集字作品

```
使用者搜尋字圖
  │
  ├── 依序點選每個字的候選字圖
  │     └── 前端記錄 selectedGlyphs: { [char]: { glyphId, char, position } }
  │
  ├── 填寫作品標題、選擇顯示方向、設定可見性
  │
  ├── 點擊「儲存集字」
  │     └── POST /api/collections
  │           ├── 驗證：需要登入
  │           ├── 驗證：標題不可為空
  │           ├── 驗證：至少選一個字圖
  │           ├── 建立 collections 記錄
  │           ├── 批次插入 collection_items 記錄
  │           └── 同步資料庫至 Blob（如有設定）
  │
  └── 重導向至作品詳情頁 /collections/[id]
```

### 3.2 瀏覽集字作品

- **個人作品**：`/me` 頁面，顯示自己的所有集字作品
- **公開作品列表**：`/collections` 頁面，顯示所有公開作品
- **作品詳情**：`/collections/[id]`，顯示完整集字結果

### 3.3 更新集字作品

- 僅作品所有者可更新
- 可更新：標題、顯示方向、可見性
- 無法更換已選取的字圖（需刪除後重建）

### 3.4 刪除集字作品

- 僅作品所有者可刪除
- 刪除時 `collection_items` 記錄隨之串聯刪除（CASCADE）

---

## 4. API 規格

### 4.1 GET /api/collections
**用途**：列出當前使用者的集字作品

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要登入 |
| 回應格式 | 集字作品陣列（含每件作品的字圖列表） |

### 4.2 POST /api/collections
**用途**：建立新集字作品

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要登入 |

**請求 Body：**
```json
{
  "title": "作品標題",
  "text": "集字文字內容",
  "displayDirection": "horizontal",
  "visibility": "public",
  "items": [
    { "glyphId": 1, "char": "山", "position": 0 },
    { "glyphId": 5, "char": "水", "position": 1 }
  ]
}
```

**成功回應（201 Created）：**
```json
{
  "id": 42,
  "title": "作品標題",
  "text": "山水",
  "displayDirection": "horizontal",
  "visibility": "public",
  "created_at": "2025-01-01T00:00:00Z"
}
```

**錯誤回應：**

| HTTP 狀態碼 | 情境 |
|------------|------|
| 401 | 未登入 |
| 400 | 標題為空 / items 為空陣列 / 字圖 ID 不存在 |

### 4.3 GET /api/collections/[id]
**用途**：取得單一集字作品詳情

| 項目 | 內容 |
|------|------|
| 認證需求 | 不需要（但私密作品需為所有者） |

**成功回應（200 OK）：**
```json
{
  "id": 42,
  "title": "作品標題",
  "text": "山水",
  "displayDirection": "horizontal",
  "visibility": "public",
  "user_id": "google_user_id",
  "user_name": "王小明",
  "created_at": "2025-01-01T00:00:00Z",
  "items": [
    {
      "id": 1,
      "collection_id": 42,
      "glyph_id": 1,
      "position": 0,
      "char": "山",
      "glyph": { "id": 1, "char": "山", "author": "王羲之", ... }
    }
  ]
}
```

**錯誤回應：**

| HTTP 狀態碼 | 情境 |
|------------|------|
| 404 | 作品不存在 |
| 403 | 嘗試存取他人私密作品 |

### 4.4 PATCH /api/collections/[id]
**用途**：更新集字作品

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要登入（且為作品所有者） |

**請求 Body（部分更新）：**
```json
{
  "title": "新標題",
  "displayDirection": "vertical",
  "visibility": "private"
}
```

### 4.5 DELETE /api/collections/[id]
**用途**：刪除集字作品

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要登入（且為作品所有者） |
| 成功回應 | 204 No Content |
| 串聯刪除 | collection_items 隨之刪除 |

---

## 5. 顯示方向

| 值 | 中文名稱 | 說明 |
|-----|---------|------|
| `horizontal` | 橫排 | 字圖由左至右排列 |
| `vertical` | 直排 | 字圖由上至下排列（傳統中文書法格式） |

---

## 6. 可見性控制

| 值 | 說明 | 存取限制 |
|-----|------|---------|
| `public` | 公開 | 所有人可瀏覽 |
| `private` | 私密 | 僅作品所有者可瀏覽 |

---

## 7. 前端組件

| 組件 | 用途 |
|------|------|
| `CollectionGlyphDisplay.tsx` | 集字作品展示（橫排/直排切換） |
| `app/me/page.tsx` | 個人集字作品列表管理 |
| `app/collections/page.tsx` | 公開作品列表 |
| `app/collections/[id]/page.tsx` | 作品詳情頁 |

---

## 8. 業務規則

1. **每字僅能選一個字圖**：同一位置的字只能對應一個字圖
2. **位置順序**：`position` 從 0 開始，代表文字在整件作品中的位置
3. **字圖不可用情境**：若集字作品引用的字圖被刪除（CASCADE），集字項目也會隨之刪除
4. **登入才可建立**：訪客只能瀏覽公開作品
