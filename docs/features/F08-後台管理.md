# F08 — 後台管理

> 功能類別：系統管理
> 相關檔案：`app/admin/`、`app/api/admin/`、`middleware.ts`

---

## 1. 功能概述

後台管理功能專供管理員使用，提供字圖資料庫管理、批量上傳、系統統計、安全事件監控等功能。管理員身份由環境變數 `MOSCRIPT_ADMIN_EMAILS` 白名單控制，透過 `middleware.ts` 保護所有 `/admin/*` 路由。

---

## 2. 存取控制

| 保護機制 | 說明 |
|---------|------|
| Middleware 層 | `middleware.ts` 攔截所有 `/admin/*` 請求 |
| Session 驗證 | 需要有效的加密 Session Cookie |
| 管理員驗證 | `isAdmin` 需為 true（email 在白名單中） |
| 未授權處理 | 未登入 → 重導向至首頁；已登入但非管理員 → 403 回應 |

---

## 3. 後台功能清單

### 3.1 系統統計（/admin）

| 統計項目 | 說明 |
|---------|------|
| 字圖總數 | 系統書庫字圖數量 |
| 書體分佈 | 各書體的字圖數量統計 |
| 熱門搜尋 | 最常搜尋的字（來自 usage_events） |
| 使用者數 | 已註冊使用者總數 |
| 近期上傳 | 最新上傳的字圖列表 |

### 3.2 字圖管理

| 操作 | 說明 |
|------|------|
| 建立字圖 | POST /api/glyphs（系統書庫字圖，owner_user_id = NULL） |
| 編輯字圖 | PATCH /api/glyphs/[id]（可修改任何欄位） |
| 刪除字圖 | DELETE /api/glyphs/[id]（含實體檔案） |
| 批量上傳 | 透過 /admin/upload 頁面，一次上傳多張字圖 |

### 3.3 批量上傳（/admin/upload）

管理員專用的批量字圖上傳功能：

```
管理員至 /admin/upload 頁面
  │
  ├── 選擇多個圖片檔案
  ├── 填寫共用元資料（書體、授權類型等）
  │
  └── 提交 POST /api/admin/upload
        ├── 驗證管理員身份
        ├── 逐一處理每個圖片
        │     ├── 從檔案名解析元資料（若有命名規範）
        │     └── 儲存至 public/glyphs/（系統字圖）
        ├── 批次寫入 glyphs 表（source = 'import'）
        ├── 記錄 admin_audit_log（action = glyph_batch_upload）
        └── 同步資料庫至 Blob
```

### 3.4 安全監控（/api/admin/security）

查看近期安全事件日誌：

- OAuth State 不符（疑似 CSRF 攻擊）
- 速率限制觸發（疑似爬蟲或濫用）
- API 未授權存取嘗試
- 資源存取違規

## 4. API 規格

### 4.1 GET /api/admin/stats
**用途**：取得系統統計資訊

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |

**成功回應（200 OK）：**
```json
{
  "glyphCount": 5000,
  "userCount": 150,
  "scriptDistribution": [
    { "script_type": "行", "count": 2000 },
    { "script_type": "草", "count": 1500 }
  ],
  "topChars": [
    { "char": "山", "count": 45 },
    { "char": "水", "count": 38 }
  ],
  "recentUploads": [...]
}
```

### 4.2 POST /api/admin/upload
**用途**：後台批量上傳字圖

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |
| Content-Type | `multipart/form-data` |

**表單欄位：**

| 欄位 | 說明 |
|------|------|
| `files` | 多個圖片檔案 |
| `scriptType` | 書體（選填，共用） |
| `license` | 授權類型（選填） |
| `source` | 來源標記（預設 `import`） |

**成功回應（200 OK）：**
```json
{
  "uploaded": 15,
  "failed": 2,
  "errors": [
    { "filename": "test.jpg", "reason": "缺少中文字元信息" }
  ]
}
```

### 4.3 GET /api/admin/security
**用途**：取得安全事件日誌

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |

**Query 參數：**

| 參數 | 說明 | 預設值 |
|------|------|--------|
| `severity` | 嚴重程度篩選（high/medium/low） | 全部 |
| `limit` | 回傳筆數 | 100 |
| `offset` | 分頁偏移 | 0 |

### 4.4 POST /api/glyphs（管理員建立）
**用途**：建立系統書庫字圖

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |

**請求 Body：**
```json
{
  "char": "山",
  "author": "王羲之",
  "scriptType": "行",
  "workTitle": "蘭亭集序",
  "imageUrl": "https://...",
  "source": "manual",
  "license": "non-commercial-research",
  "qualityScore": 5
}
```

### 4.5 PATCH /api/glyphs/[id]（管理員編輯）
**用途**：編輯任意字圖元資料

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |

**可編輯欄位：**
- `char`、`author`、`script_type`、`work_title`
- `source`、`license`、`quality_score`
- `visibility`

### 4.6 DELETE /api/glyphs/[id]（管理員刪除）
**用途**：刪除任意字圖

| 項目 | 內容 |
|------|------|
| 認證需求 | 需要管理員 |
| 副作用 | 刪除實體檔案、串聯刪除 likes 與 collection_items |

---

## 5. 管理員操作日誌

所有後台操作均記錄至 `admin_audit_logs` 表：

| action 值 | 操作說明 |
|-----------|---------|
| `glyph_create` | 建立字圖 |
| `glyph_update` | 更新字圖 |
| `glyph_delete` | 刪除字圖 |
| `glyph_batch_upload` | 批量上傳字圖 |

|------|------|
| `/admin` | 後台主頁（統計資訊 Dashboard） |
| `/admin/upload` | 批量上傳字圖頁面 |
