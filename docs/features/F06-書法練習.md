# F06 — 書法練習

> 功能類別：教育輔助功能
> 相關檔案：`app/practice/[id]/page.tsx`、`components/GlyphPracticeCanvas.tsx`

---

## 1. 功能概述

書法練習功能讓使用者參照字圖，在瀏覽器上直接以滑鼠或觸控螢幕進行書法練習。系統以 HTML5 Canvas 繪製自然筆跡（使用 `perfect-freehand` 函式庫模擬毛筆效果），背景顯示參考字圖，使用者可清除畫布重複練習。

---

## 2. 功能特點

| 特點 | 說明 |
|------|------|
| 自然筆跡 | 使用 `perfect-freehand` 根據筆壓/速度模擬毛筆粗細變化 |
| 參考字圖 | Canvas 背景顯示半透明的書法字圖供臨摹 |
| 觸控支援 | 支援滑鼠（Mouse Events）與觸控（Touch Events） |
| 清除功能 | 一鍵清除畫布，重新練習 |
| 無需登入 | 任何人可使用（對公開字圖） |

---

## 3. 頁面路由

```
/practice/[id]
```

- `[id]`：字圖的 `glyphs.id`
- 頁面載入時取得字圖資料（`GET /api/glyphs/[id]`）
- 私密字圖需通過存取控制驗證

---

## 4. 畫布規格

| 項目 | 規格 |
|------|------|
| 技術 | HTML5 Canvas 2D Context |
| 筆跡函式庫 | `perfect-freehand` v1.2.3 |
| 輸入裝置 | 滑鼠、觸控（單指） |
| 背景圖 | 字圖縮圖（`thumbnail_url`）或完整圖（`image_url`），半透明顯示 |
| 畫布尺寸 | 響應式，隨容器大小調整 |

---

## 5. 操作說明

| 操作 | 效果 |
|------|------|
| 按住滑鼠/觸控拖移 | 繪製筆跡 |
| 放開滑鼠/觸控 | 完成一筆 |
| 點擊「清除」按鈕 | 清空畫布上的筆跡（背景字圖保留） |
| 點擊「回上一筆」（若有） | 撤銷最後一筆 |

---

## 6. 進入練習頁的方式

| 入口 | 說明 |
|------|------|
| 字圖詳情頁 `/glyph/[id]` | 點擊「開始練習」按鈕 |
| 直接輸入網址 | `/practice/[id]` |

---

## 7. 存取控制

- **公開字圖**：任何人可進入練習頁
- **私密字圖**：僅字圖所有者可進入（API 層驗證）
- 非法存取回傳 403，前端重導向至首頁
