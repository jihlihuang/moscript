# 環境變數完整說明

> 本文列出系統所有環境變數，說明用途、必填性與範例值。
> 設定方式：本機開發使用 `.env.local`，正式部署設定於 Azure Container App 或 CI/CD。

---

## 快速分類

| 類別 | 必填 | 說明 |
|------|------|------|
| [Google OAuth](#1-google-oauth) | 是 | 登入功能必備 |
| [Session 加密](#2-session-加密) | 是 | 維持登入狀態必備 |
| [應用程式 URL](#3-應用程式-url) | 是 | OAuth 回調與連結生成 |
| [管理員設定](#4-管理員設定) | 建議 | 未設定則無人可進後台 |
| [Azure Blob Storage](#5-azure-blob-storage) | 否 | 僅雲端部署需要 |
| [資料目錄](#6-資料目錄路徑) | 否 | 本機預設即可 |
| [Azure 部署腳本](#7-azure-部署腳本用) | 否 | 僅執行部署腳本時需要 |

## 1. Google OAuth

### `GOOGLE_CLIENT_ID`

| 項目 | 內容 |
|------|------|
| 必填 | 是 |
| 來源 | Google Cloud Console → 憑證 → OAuth 2.0 用戶端 ID |
| 格式 | `<數字>.apps.googleusercontent.com` |
| 範例 | `123456789012-abcdefghijklmnop.apps.googleusercontent.com` |
| 用途 | Google OAuth 登入流程的 Client ID |

### `GOOGLE_CLIENT_SECRET`

| 項目 | 內容 |
|------|------|
| 必填 | 是 |
| 來源 | Google Cloud Console → 同上 |
| 格式 | `GOCSPX-` 開頭的字串 |
| 範例 | `GOCSPX-abcdefghijklmnopqrstuvwx` |
| 用途 | Google OAuth 登入流程的 Client Secret |
| 安全注意 | 絕對不可提交至 Git，不可出現在前端程式碼 |

---

## 2. Session 加密

### `AUTH_SECRET`

| 項目 | 內容 |
|------|------|
| 必填 | 是 |
| 格式 | 64 字元十六進位字串（32 bytes） |
| 產生方式 | `openssl rand -hex 32` |
| 範例 | `a3f8b2c1d4e5f6a7b8c9d0e1f2a3b4c5d6e7f8a9b0c1d2e3f4a5b6c7d8e9f0a1` |
| 用途 | AES-256-GCM 加密 Session Cookie 的金鑰 |
| 安全注意 | 更換此金鑰會使所有現有 Session 失效（所有使用者需重新登入） |

---

## 3. 應用程式 URL

### `MOSCRIPT_APP_URL`

| 項目 | 內容 |
|------|------|
| 必填 | 是 |
| 格式 | 完整 URL，不含結尾斜線 |
| 本機範例 | `http://localhost:3000` |
| 正式範例 | `https://moscript.azurecontainerapps.io` |
| 用途 | Google OAuth 回調 URL 組合（`/api/auth/google/callback`）、分享連結生成 |
| 注意 | 此值需與 Google Cloud Console 的「已授權的重新導向 URI」一致 |

---

## 4. 管理員設定

### `MOSCRIPT_ADMIN_EMAILS`

| 項目 | 內容 |
|------|------|
| 必填 | 否（未設定則無管理員） |
| 格式 | 逗號分隔的 email 清單 |
| 範例 | `admin@example.com,dev@example.com` |
| 用途 | 判斷使用者是否具有管理員權限（不使用資料庫，直接比對 email） |
| 注意 | 新增/移除管理員只需更新此環境變數，無需修改資料庫；需重啟應用程式才生效 |

---

## 5. Azure Blob Storage

> 以下四個變數為一組，需同時設定才能啟用 Blob Storage。若未設定，字圖與資料庫改為本機儲存。

### `AZURE_STORAGE_CONNECTION_STRING`

| 項目 | 內容 |
|------|------|
| 必填 | 否（雲端部署建議設定） |
| 格式 | `DefaultEndpointsProtocol=https;AccountName=...;AccountKey=...;EndpointSuffix=core.windows.net` |
| 取得方式 | `az storage account show-connection-string --name <account> --resource-group <rg> --query connectionString --output tsv` |
| 用途 | Azure Blob Storage 連線，啟用此功能後字圖儲存至 Blob 而非本機 |
| 安全注意 | 正式環境應以 Secret 儲存（`az containerapp secret set`），不可硬編碼 |

### `AZURE_STORAGE_CONTAINER_NAME`

| 項目 | 內容 |
|------|------|
| 必填 | 是（若啟用 Blob） |
| 範例 | `moscript` |
| 用途 | Blob Storage 容器名稱 |

### `MOSCRIPT_GLYPHS_PREFIX`

| 項目 | 內容 |
|------|------|
| 必填 | 否 |
| 預設值 | `glyphs` |
| 範例 | `glyphs` |
| 用途 | 公開字圖在 Blob 中的路徑前綴（`glyphs/山/王羲之_行_001.jpg`） |

### `MOSCRIPT_PRIVATE_GLYPHS_PREFIX`

| 項目 | 內容 |
|------|------|
| 必填 | 否 |
| 預設值 | `private-glyphs` |
| 範例 | `private-glyphs` |
| 用途 | 私密字圖在 Blob 中的路徑前綴 |

### `MOSCRIPT_DB_BLOB_NAME`

| 項目 | 內容 |
|------|------|
| 必填 | 否 |
| 預設值 | `data/moscript.sqlite` |
| 範例 | `data/moscript.sqlite` |
| 用途 | SQLite 資料庫在 Blob 中的路徑；容器啟動時從此路徑下載資料庫 |

---

## 6. 資料目錄路徑

### `MOSCRIPT_DATA_DIR`

| 項目 | 內容 |
|------|------|
| 必填 | 否 |
| 預設值 | `data`（相對於專案根目錄） |
| 容器環境範例 | `/tmp/moscript-data` |
| 用途 | SQLite 資料庫與私密字圖的本機存放目錄 |
| 容器注意 | Azure Container App 的根目錄為唯讀，需設定為可寫路徑（如 `/tmp/moscript-data`） |

## 7. Azure 部署腳本用

> 以下變數僅供 `npm run deploy:azure` 與 `npm run restart:azure` 腳本使用，不影響應用程式運行。

### `AZURE_RESOURCE_GROUP`

| 項目 | 內容 |
|------|------|
| 範例 | `moscript-rg` |
| 用途 | Azure 資源群組名稱 |

### `AZURE_CONTAINER_APP_NAME`

| 項目 | 內容 |
|------|------|
| 範例 | `moscript` |
| 用途 | Azure Container App 名稱 |

### `AZURE_ACR_NAME`

| 項目 | 內容 |
|------|------|
| 範例 | `moscriptacr2026` |
| 用途 | Azure Container Registry 名稱 |

### `AZURE_IMAGE_REPOSITORY`

| 項目 | 內容 |
|------|------|
| 範例 | `moscript` |
| 預設值 | `moscript` |
| 用途 | Docker Image 名稱（Repository 部分） |

### `DOCKER_PLATFORM`

| 項目 | 內容 |
|------|------|
| 範例 | `linux/amd64` |
| 預設值 | `linux/amd64` |
| 用途 | Docker build 目標平台（Apple Silicon 開發時仍需指定 amd64） |

### `AZURE_BUILD_MODE`

| 項目 | 內容 |
|------|------|
| 可選值 | `acr`（雲端建置）、`local`（本機建置） |
| 預設值 | `acr` |
| 用途 | 選擇 Docker build 方式；建議 Apple Silicon Mac 使用 `acr` |

## 最小本機開發設定（`.env.local` 範例）

```bash
# 必填
GOOGLE_CLIENT_ID=xxx.apps.googleusercontent.com
GOOGLE_CLIENT_SECRET=GOCSPX-xxx
AUTH_SECRET=（openssl rand -hex 32 的輸出）
MOSCRIPT_APP_URL=http://localhost:3000

# 強烈建議
MOSCRIPT_ADMIN_EMAILS=your@gmail.com

# Azure Blob（本機開發可不填，字圖存本機）
# AZURE_STORAGE_CONNECTION_STRING=
# AZURE_STORAGE_CONTAINER_NAME=
```
