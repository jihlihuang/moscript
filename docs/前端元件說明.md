## GlyphImage

**檔案：** [components/GlyphImage.tsx](../components/GlyphImage.tsx)
**用途：** 統一處理公開字圖（靜態路徑）與私密字圖（`/api/glyphs/[id]/asset`）的圖片顯示，對外暴露相同介面。

**Props：**

| Prop | 類型 | 必填 | 說明 |
|------|------|------|------|
| `glyph` | GlyphRow | 是 | 字圖資料物件 |
| `className` | string | 否 | Tailwind CSS class |
| `useThumbnail` | boolean | 否 | 是否優先使用縮圖 URL |

**使用場景：**
- 搜尋結果列表（使用縮圖）
- 字圖詳情頁（使用原圖）
- 集字展示（使用原圖）

**私密字圖處理邏輯：**
```
image_url 包含 '/api/glyphs/' → 直接使用（帶認證 Cookie 的 fetch）
image_url 以 '/glyphs/' 開頭  → 靜態路徑，直接 <img src>
image_url 以 'https://'開頭  → Azure Blob URL，直接 <img src>
```

## GlyphLikeButton

**檔案：** [components/GlyphLikeButton.tsx](../components/GlyphLikeButton.tsx)
**用途：** 字圖按讚按鈕，包含樂觀更新（點擊立即更新 UI，等待 API 確認）與錯誤回滾。

**Props：**

| Prop | 類型 | 必填 | 說明 |
|------|------|------|------|
| `glyphId` | number | 是 | 字圖 ID |
| `initialLiked` | boolean | 是 | 初始按讚狀態 |
| `initialCount` | number | 是 | 初始按讚數 |
| `isLoggedIn` | boolean | 是 | 使用者是否已登入 |

**使用場景：** 字圖詳情頁 `/glyph/[id]`

---

## GlyphPracticeCanvas

**檔案：** [components/GlyphPracticeCanvas.tsx](../components/GlyphPracticeCanvas.tsx)
**用途：** 書法練習主元件，以 HTML5 Canvas 實作可繪圖的練習畫布，背景顯示字圖供臨摹，使用 `perfect-freehand` 模擬自然筆跡。

**Props：**

| Prop | 類型 | 必填 | 說明 |
|------|------|------|------|
| `glyph` | GlyphRow | 是 | 作為背景的字圖 |
| `backgroundOpacity` | number | 否 | 背景字圖透明度（0~1） |

**輸入支援：** Mouse Events、Touch Events（單指）

**使用場景：** 書法練習頁 `/practice/[id]`

**依賴套件：** `perfect-freehand`（自然筆跡演算法）

---

## PersonalGlyphManager

**檔案：** [components/PersonalGlyphManager.tsx](../components/PersonalGlyphManager.tsx)
**用途：** 個人上傳字圖的清單管理介面，支援選取、批量刪除、批量切換可見性。

**使用場景：** 個人頁面 `/me`（「我的字圖」分頁）

**主要功能：**
- 顯示個人上傳的所有字圖（含 like_count、collection_count）
- 多選模式（checkbox）
- 批量刪除（確認對話框）
- 批量切換 public/private

## CollectionGlyphDisplay

**檔案：** [components/CollectionGlyphDisplay.tsx](../components/CollectionGlyphDisplay.tsx)
**用途：** 集字作品的完整展示，支援橫排（horizontal）與直排（vertical）兩種排列方式。

**Props：**

| Prop | 類型 | 必填 | 說明 |
|------|------|------|------|
| `collection` | Collection & { items: CollectionItemWithGlyph[] } | 是 | 含字圖的集字作品 |
| `direction` | `'horizontal'` \| `'vertical'` | 否 | 覆蓋排列方向 |

**使用場景：** 集字作品詳情頁 `/collections/[id]`

## useCollectionDirectionPreference（Hook）

**檔案：** [components/useCollectionDirectionPreference.ts](../components/useCollectionDirectionPreference.ts)
**用途：** 記憶使用者對集字排列方向（橫排/直排）的偏好，儲存於 `localStorage`，下次開啟時自動套用。

**回傳：**
```typescript
{
  direction: 'horizontal' | 'vertical',
  setDirection: (d: 'horizontal' | 'vertical') => void
}
```

**儲存 key：** `moscript-collection-direction`
